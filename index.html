About

Halo, semuanya!

Perkenankan saya menceritakan menceritakan tentang diri saya. Nama saya Aditya Athawidhahmeisa atau nama di game adalah Kagura Aoi. Saya lahir pada tanggal 7 Mei 2001 di Pontianak, Kalimantan Barat. Saya sekarang berusia 21 tahun.

Saat ini, saya berkuliah di jurusan Teknik Informatika di Pontianak. Sejak awal semester 1, saya mulai mengenal pemrograman dan tertarik pada bidang pemrograman web. Biasanya, saya menghabiskan waktu dengan ngoding atau bermain game. Mimpi saya adalah menjadi programmer dan entrepreneur sehingga nantinya saya bisa travel destinasi-destinasi yang ada di Indonesia.

Saat ini, saya sedang menekuni skill saya pada web design. Alasan saya tertarik pada bidang pemrograman web yakni saya suka dengan tampilan-tampilan website yang berwarna dan elegan. Oleh karena itu, saya juga tertarik untuk mempelajari hal tersebut.
